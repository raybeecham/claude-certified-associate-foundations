# Claude Foundations Study Guide

This site is the starting point for an independent, scenario-driven study repository for the **Claude Certified Associate - Foundations** certification.

The complete repository includes seven domain modules, labs, flashcards, quizzes, practice exams, prompt templates, and an offline study CLI.

## Start here

1. Read the [certification overview](certification-overview.md).
2. Select a schedule in the [four-week study plan](study-plan.md).
3. Review the [exam strategy](exam-strategy.md).
4. Begin [Module 1 in the GitHub repository](https://github.com/raybeecham/claude-certified-associate-foundations/tree/main/modules/01-platform-model-foundations).
5. Record results in the [progress tracker](progress-tracker.md).

## Core study principle

Do not begin by memorizing prompt patterns. Begin with the task, risk, evidence, and success criteria. Then select the product surface, model characteristics, prompt structure, tools, evaluation method, human controls, and operational safeguards that fit the use case.
