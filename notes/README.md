# Personal Notes

This directory is intentionally empty except for this guide.

Use a private branch or local, ignored files for personal notes. Do not commit:

- employer-confidential information;
- API keys or credentials;
- proprietary preparation-course content;
- recalled live-exam questions;
- personal study records you do not want public; or
- controlled technical data.

Suggested local files:

```text
notes/my-weak-areas.md
notes/my-lab-results/
notes/my-source-checks.md
```

Add personal paths to `.git/info/exclude` or your global Git ignore if they should never be committed.
