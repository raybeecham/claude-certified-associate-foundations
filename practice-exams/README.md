# Practice Exams

The repository includes two balanced, original practice exams. Each exam contains **28 questions**, four from each study domain.

## Recommended conditions

- Closed notes
- One uninterrupted sitting
- No web search
- Record start and finish time
- Mark uncertain items without changing them immediately
- Review every distractor afterward

These practice scores are not an official predictor of certification results.

## Review method

Classify each miss:

- **Knowledge:** The concept was not known.
- **Reading:** A material clue was overlooked.
- **Judgment:** Two plausible controls were not prioritized correctly.
- **Execution:** The answer changed without evidence, time was mismanaged, or the selected option did not match the intended choice.

For each miss, write one scenario variation that would make a distractor correct.

## Suggested interpretation

This is an internal study scale, not an official passing standard.

| Score | Suggested action |
|---:|---|
| 90–100% | Review only weak objectives and volatile facts |
| 80–89% | Target missed domains, then retest |
| 70–79% | Repeat module labs and quizzes before retest |
| Below 70% | Rebuild domain notes and delay full-exam repetition |

Regardless of percentage, remediate every critical miss involving privacy, unauthorized access, unsupported high-impact advice, prompt injection, or unsafe side effects.
